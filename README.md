Repositório criado para a cadeira de Algoritmos e Programação B do 2º semestre da graduação em Ciência da Computação, 2025/2.

Este repositório contém os principais materiais desenvolvidos durante a disciplina. Alguns conteúdos adicionais não foram incluídos pois foram produzidos antes da organização deste repositório.

Tópicos abordados:

- Unidade 1 - Conjuntos heterogêneos
Criar estruturas de dados heterogêneas utilizadas em programação de maneira útil.

- Unidade 2 - Modularização
Aplicar os diferentes tipos de modularização na construção de programas de computador de maneira adequada.

- Unidade 3 - Tratamento de arquivos
Criar programas computacionais que persistam dados em memória secundária de maneira útil.

- Unidade 4 - Ponteiros
Compreender conceitos relacionados à alocação dinâmica para solução computacional de problemas do mundo real de maneira precisa.
